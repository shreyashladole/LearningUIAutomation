About tTab

 

A simple new tab, with Yahoo provided search.

tTab is a convenient new tab page that makes your internet search much easier.

Open a new tab, type your query in the search box, click "Enter" and get your Yahoo search results.

The extension doesn't handle any sensitive or personal user data.

 

The extension requires the following permissions:

"tabs" - This permission is necessary for replacing the page you see when opening a new tab.

"*://*.yahoo.com/" - This permission is necessary for providing you with Yahoo's search suggestions.

"contextMenus" - This permission is necessary for an easy access to this "About" page.
